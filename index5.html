Project Overview

To create a place for cartographers to share thier projects.

Project Details

In the mapping community we tend not to work collaboratively. This project is a simple way to break down those silos and get more collaboration. All too often a map is made for a particular project or class work and three people see it. But with Mapping the Bay Area other researchers will have access to a repositioty if excellent mapping projects that have already been made and collaborate with the original author, possibly adding more data and information to enhace the reach of the map.

Project Overview

Oakland Ring Map of Social Services

Project Details

This project was deisgned with the explicit purpose of developing a method to show the different concentrations of the social service clients in the City of Oakland. The ability to display a number of variables into one comprehisive map has troubled geographic scientists for decades. Using a ring technique we were able to present all of the information in a single map, which is easy for the audience to meaningfully interpret.


Project Overview

The OSNI BRT Line

Project Details

This interactive map was designed to examine the potential impact of the Bus Rapid Transit Line (BRT) in East Oakland, CA. It was created for The Oakland Sustainable Neighborhood Initiative (OSNI), which was convened to examine the impact of the purposed BRT line along the International Boulevard, between Lake Merritt and 109th Avenue. This resource goes beyond the typical paper maps and allows the user to interact with the information and understand the different variables, such as, land use, ownership, and crime that capture some of the on-the-ground phenomenon.
